🐙 GitHub Sync
Backup data otomatis ke GitHub repo kamu. Setup sekali, klik sync kapanpun.
📋
Cara setup (sekali saja):
1. Buat repo baru di GitHub (bisa private) — contoh: my-financial-data
2. Buka github.com → Settings → Developer settings → Personal access tokens → Tokens (classic)
3. Generate token baru → centang scope repo → Copy token
4. Isi form di bawah → Simpan
✨ AI Import — API Key Anthropic
Diperlukan untuk fitur AI Import. Daftar gratis di console.anthropic.com → API Keys
